# 實戰kubernetes課程指南

- **第一篇** 先把容器與 Kubernetes 的世界觀建立起來,從虛擬化一路走到你的第一座叢集。
- **第二篇** 進入核心物件與宣告式模型,這是後面所有章節的地基。
- **第三篇** 處理服務、網路與流量治理:Service、DNS、Ingress、CNI 與 Network Policy。
- **第四篇** 拆解儲存與有狀態應用:Volume、PV/PVC、StatefulSet、ConfigMap 與 Secret。
- **第五篇** 談應用生命週期與自動化維運:探針、部署策略、Rollout、Job/CronJob。
- **第六篇** 深入排程、資源與自動擴縮:QoS、Affinity、HPA 與節點擴縮。
- **第七篇** 完整覆蓋安全與治理:威脅模型、RBAC、Security Context 與 Admission Control。
- **第八篇** 教你多容器模式與擴充 Kubernetes:Sidecar、CRD、Operator、Helm、Kustomize。
- **第九篇** 收尾在生產級維運與生態系:監控、GitOps、Service Mesh、多雲、Serverless 與 kubectl 精通。
